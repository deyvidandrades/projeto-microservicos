Projeto de Microserviços

A proposta é de desenvolver a demonstração de uma aplicação implementada utilizando o estilo arquitetônico baseado em objetos e estilo arquitetônico de microserviços.

Serviços

Serviço de aplicacao

http://127.0.0.1:5000 {% if application %} ON {% else %} OFF {% endif %}

Serviço de usuarios

http://127.0.0.1:5001 {% if usuarios %} ON {% else %} OFF {% endif %}

Serviço de postagens

http://127.0.0.1:5002 {% if postagens %} ON {% else %} OFF {% endif %}

Serviço de comentarios

http://127.0.0.1:5003 {% if comentarios %} ON {% else %} OFF {% endif %}

Endpoints do Serviço de usuários

Carregando a lista de todos os usuários:

"/usuarios"

Carregando um usuário pelo id:

"/usuarios/{string:user_id}"

Carregando o perfil completo de um usuário pelo id.

"/usuarios/{string:user_id}/perfil"

Esse endpoint retorna o perfil do usuário com a adição das postagens feitas.
